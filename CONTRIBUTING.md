# Contributing · 参与共建

Thank you for helping make this open library of Chinese traditional patterns
more accurate, more complete, and more useful! 🙏

感谢你愿意一起把这份中国传统纹样开放图库做得更准、更全、更好用！

---

## Ways to contribute · 你可以怎么参与

### 🔍 Fix the research · 指正考据
The **period** and **region** fields are compiled from public sources and are
*indicative references, not strict dating*. If you spot an error or can add a
better-sourced fact (museum collection, archaeological report, scholarly work),
please open an Issue with the pattern `id` and your source.

时期 / 地域为综合公开资料整理，属指示性参考。若发现错误或有更可靠来源（馆藏、
考古报告、学术文献），欢迎带上纹样 `id` 与出处开 Issue。

### 🌏 Improve translations · 改进翻译
Polish the English copy, or help with the Japanese / Korean READMEs and (future)
per-pattern translations.

润色英文文案，或协助日 / 韩 README 及未来逐条纹样翻译。

### 🎨 Contribute patterns · 贡献纹样
Submit new motifs to help grow toward 800. Submissions **must be original
re-designs** (not copies of copyrighted artworks or museum photographs) and
follow the existing data format.

提交新纹样，助力扩充至 800 条。提交内容**须为原创再设计**（不得复制受版权保护的
作品或博物馆照片），并遵循现有数据格式。

### 💻 Improve tooling · 改进工具
Data validation, export scripts (SVG, palette files…), visualisations, etc.

数据校验、导出脚本（SVG、色卡…）、可视化等。

---

## Data format · 数据格式

See the **Data format** section in the [README](README.en.md#-data-format).
Each pattern lives in `patterns/<category>/<id>_<name>/` with three files:
`<id>_<name>.png`, `<id>_<name>.md`, `<id>_<name>.json`, and is also indexed in
`data/patterns.json` / `data/patterns.csv`.

---

## How to submit · 如何提交

1. **Open an Issue first** for anything non-trivial, so we can discuss it.
2. Fork → create a branch → make your change → open a Pull Request.
3. Keep PRs focused (one topic per PR) and describe what & why.

对较大改动请**先开 Issue 讨论**；Fork → 建分支 → 修改 → 提 PR；保持每个 PR 聚焦单一主题。
