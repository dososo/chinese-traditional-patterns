# 049 万字锦纹 Wan Character Brocade Pattern

## 一句话简介
以万字纹为核心连续铺陈，象征万福绵延、繁荣昌盛。

## 纹样简介
万字锦纹以“万”字式吉祥符号为基础，与花卉、云纹或锦地结合形成连续锦纹。它常用于织物、地毯和器物装饰中，兼具吉祥象征与装饰节奏感。

## 纹样构成
以万字形单元反复连续排列，可横向、斜向或网格化组织，并加入花形节点增强装饰性。

## 视觉特征
- 符号重复，识别度高
- 纹面饱满，适合铺陈
- 兼具几何感和吉祥感
- 常与锦地花纹结合

## 文化寓意
寓意万福绵长、繁盛昌隆、吉祥延续。

## 常见载体
- 织锦
- 地毯
- 壁饰
- 服饰
- 包装底纹
- 文博衍生设计

## 色彩建议
- 青蓝 + 古金：典雅锦纹感
- 墨绿 + 金色：华丽稳重
- 绛红 + 米金：礼品包装感

## 现代应用
适合大面积底纹、产品包装、网页背景、文化空间软装和衍生品印花。
