# 078 双钱纹 Double Coin Pattern

## 一句话简介
以两枚古钱相连构成，寓意财运相通、吉庆连连与圆融和合。

## 纹样简介
双钱纹由两枚圆形方孔钱币相连而成，是典型的吉祥器物纹样。相比单个铜钱纹，双钱纹更强调连接、成双与财运相通。

## 纹样构成
以两枚外圆内方的古钱为基本单元，常以短横连接，也可重复排列成连续图案。

## 视觉特征
外圆内方、结构对称、识别度高，既可做主图也可做连续底纹。

## 文化寓意
寓意财富、连通、成双成对、圆满与吉庆。

## 常见载体
织物、瓷器、家具雕饰、年节装饰、包装图案、文创配饰。

## 色彩建议
- 古金 + 墨绿 + 米白
- 朱砂 + 金色 + 暗青
- 青铜色 + 暖白 + 深绿

## 现代应用
适合用于财运主题礼品、年节包装、银行或商业文创、吉祥挂件和图标系统。
